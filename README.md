# Internship Tracker

A Kanban board to track internship applications.

## Features
- Add internships with company name, role and date
- Drag and drop cards between stages
- Delete cards
- Data persists after page refresh (localStorage)

## Stages
Applied → Shortlisted → Offer → Selected → Rejected

## Built with
HTML, CSS, JavaScript

## How to run
Just open index.html in your browser
